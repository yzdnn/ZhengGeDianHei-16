## **～项目概述～**

「正格点黑 16」是基于「ドットゴシック16」增补简繁体汉字的16px（含字间距）点阵黑体，适合在各种像素风场景使用。

熟悉像素字体/点阵字体的朋友都知道，日韩厂家的汉字点阵与我们常见的国标点阵的风格有所不同。（例如MS Gothic和Dotum的内嵌位图）

国标点阵有种宋黑折衷的风格，而日韩厂家做点阵字库时没有往宋体（明朝体）靠的习惯。

蒙纳（原Fontworks）的ドットゴシック16（DotGothic16）便是其中之一。

前几年这款字体在Google Fonts上和Fontworks的其他几款字体开始以SIL Open Font License授权发布，相当于开放了修改权限。

看到国内还没有这种风格的简体字点阵，于是我在做完袖珍像素体后，又开了这么个坑。

经过一番处理后，我成功将字体转换成了可以编辑的.bdf点阵，开始了漫长的补字之旅。

目前本字体项目仍在持续更新，欢迎各位网友提出宝贵意见。
